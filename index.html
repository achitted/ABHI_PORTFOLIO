PROJECT TITLE: SALES TAX CALCULATOR

Description:

The sales tax calculator functions as a web application designed for determining the sales tax and total cost of a purchase. Users input the valid amount, and the calculator generates the applicable tax for the given purchase.

Technologies used:

HTML: The HTML page layout includes textboxes for user input and buttons for executing actions. Additionally, the HTML page is linked with CSS and JavaScript to enhance functionality and improve the visual appeal of the webpage.

CSS: CSS enhances the visual appeal of a webpage by employing properties to specify style attributes.

JAVASCRIPT: The JavaScript code verifies user input and executes the computation of sales tax based on the provided inputs.

PROJECT TITLE: TEST SCORE ARRAY

Description:

The web application, "Text Score Array," enables users to store test results along with names and scores. It computes the average score within a section and presents the results for each person.

Technologies used:

HTML: The HTML page collects user input, including name and score, using input fields and labels. It incorporates buttons for adding entries to the test score array, displaying results, and calculating the average score.

CSS: CSS improves the visual aesthetics of a webpage by utilizing properties to define and control various style attributes.

JAVASCRIPT: The functionality of the web page relies on the JavaScript logic, which validates user input, adds test scores to the current array, calculates the average score, and displays details about the individual with the highest scores. The display scores feature allows users to view the scores of each person.
